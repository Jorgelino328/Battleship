## How to Run

1. Clone the repository:
    ```bash
    git clone https://github.com/Jorgelino328/Battleship.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Battleship
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the server:
    ```bash
    npm start
    ```
5. Open your browser and navigate to `http://localhost:8080`.
